Getting started

Welcome

Hello there! Thanks for buying Vcard. Vcard is a Landing page template with solid design especially crafted to help you built product, apps or software Landing Page websites with huge variety of options.

This Template is suitable for all types of Product Landing Page. This template built using Bootstrap 4.5.0, Google Fonts, Font Awesome 5.15.1, Iconic Icon, Material Icons, jQuery v3.5.1, jQuery Easing v1.3, Magnific Popup - v1.1.0, Owl Carousel , ScrollPos-Styler v0.7.0 . It is highly recomended that you are familiar in a way with at least HTML5 / CSS3 / Basic JQuery.

So let's jump directly to it, and start everything from scratch.

Installation

Install the template via ftp :

  1. Connect to your website host, using some of many FTP clients, for example FileZilla.
  2. Go to your default server folder, and copy Vcard into that folder.

Setting your fonts

To use google fonts, head over to Google's Fonts and add whichever font you want to your collection. Once you finished, follow the instructions and place the generated link in custom.css file.
The link should look like this : 

@import url('https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;600;700&family=Roboto+Condensed:wght@300;400;700&display=swap');
Next, just replace the font family, with the font family you just copied. Like: 
 body {
	font-family: 'Roboto Condensed', sans-serif;
  	font-weight: 600;
   }
